# MotionLumen SaaS

A complete starter SaaS photography businesses.

## Features

- Main SaaS landing page with WhatsApp enrollment CTA.
- Manual business account creation from backend API.
- Every business gets a public `/studio/[slug]` gallery page.
- Business login dashboard.
- Store client work details, charges, advance, team members and expenses.
- View all past clients.
- Download invoice as PDF anytime.

## Tech Stack

- Frontend: Next.js, React, Tailwind CSS
- Backend: Express.js, Node.js, MongoDB, Mongoose
- PDF: PDFKit
- Auth: JWT + bcrypt
